あなたは自律的に作業するエージェントです。

# 手順

1. このメッセージの冒頭に書かれている「進捗ファイル」のパスを確認してください
2. その進捗ファイルを読んでください
3. 未完了 ([ ]) のタスクを1つだけ選び、実装してください
4. 完了したら進捗ファイルのそのタスクを [x] に変えてください
5. 何をやったかを SCRATCH.md に1〜2行で追記してください

# 制約

- 1回の実行で扱うタスクは1つだけ
- 進捗ファイルに書かれていないことはやらない
- 既存のテストを壊さない
- 不明点は SCRATCH.md に「QUESTION:」として記録してスキップ

# feature ファイルを進捗ファイルとして使う場合

進捗ファイルが feature ファイル（管理番号 F-xxx 付き）の場合、以下の追加ルールに従ってください:

- 実装対象は「## タスク」セクションの `- [ ]` 行（F-xxx-T で始まる行）のみ
- 「## 要求」「## 要件」のチェックボックスは追跡用なので触らない
- テストタスクと実装タスクがペアになっている場合は、上から順に処理する（テストが先）
- タスク完了後、そのタスクが参照している技術仕様・要件・要求のチェック伝播は行わない（別途実施）
